TACIT

The Tacit framework was created by Adam Morse, and it is designed to be lightweight and unopinionated. It provides a set of basic styles for common HTML elements such as headings, paragraphs, links, lists, and forms, as well as a responsive grid system for laying out content on different screen sizes.

One of the key features of Tacit is its use of CSS custom properties (also known as CSS variables), which allows developers to easily customize the appearance of the framework to match their design needs. It also includes a number of utility classes for common layout and styling tasks, such as centering content, hiding and showing elements, and adjusting margins and padding.

Overall, the Tacit framework is a good option for developers who want a lightweight and customizable CSS framework that doesn't impose too many design constraints.
